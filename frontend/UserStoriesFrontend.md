## User Stories Hornischer

1. Ich möchte kritische Sensorzustände sofort visuell hervorgehoben sehen, 
   damit ich bei Gefahrensituationen innerhalb von Sekunden eingreifen kann
    Akzeptanzkriterien:
    - Eine rote "critical" Badge wird angezeigt
    - Eine rote Rahmenumrandung ist sichtbar
    - Kritischer Status ist sofort im Dashboard sichtbar sein

2. Ich möchte den Gesamtzustand aller Sensoren auf einen Blick erfassen, 
   damit ich den Betriebszustand der gesamten Anlage schnell beurteilen kann
    Akzeptanzkriterien:
    - Alle verfügbaren Sensoren sind in einer Rasteransicht zu sehen
    - Für jeden Sensor sofort Wert, Einheit und Status erkennen
    - 4 Sensortypen (Temperatur, Luftfeuchtigkeit, Gas, Flamme) werden angezeigt

3. Ich möchte automatisch aktualisierte Sensordaten im 5-Sekunden-Takt erhalten, 
   damit ich kontinuierlich über Änderungen der Umgebungsbedingungen informiert bin
    Akzeptanzkriterien:
    - Es erfolgt automatisch ein neuer Datenabruf
    - Alle Sensorwerte und Status werden aktualisiert 

4. Ich möchte die genauen Grenzwerte für jeden Sensortyp einsehen können, 
   damit ich bei Alarmen sofort die richtigen Maßnahmen einleiten kann
    Akzeptanzkriterien:
    - Es ist eine klar strukturierte Übersicht aller Grenzwerte sehen
    - Es können die spezifischen Schwellenwerte nachgelesen werden
